## Deno

### 利用ライブラリ

- Flash
  - https://fresh.deno.dev/
- React

### 実行

deno.jsonのtasks(Node.jsでいうpackage.jsonのscripts)に書かれている内容に従ってタスクを実行する

```sh
// サーバー起動
deno task start
```
